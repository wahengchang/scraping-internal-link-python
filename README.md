




## 1- install
```
$ pip install
```

## 2- run 
```
$ source bin/activate
```

## freeze installed packages
```
$ pip freeze > requirements.txt

```

## Run
```
$ python main.py
```

## Remark 
Should have selenium install
```
$ docker run -d -p 4444:4444 -v /dev/shm:/dev/shm selenium/standalone-chrome:3.14.0-beryllium
```
